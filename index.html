第2章 概念学习

概念学习

定义：从有关某个布尔函数的训练样例中推断出该布尔函数；从集合的角度来看，它是实例空间X的一个子集。
归纳学习的基本假设：对未见实例最好的假设就是与训练数据最佳拟合的假设

假设空间的偏序

定义： $h_j, h_k$为在X上定义的布尔函数， 称 $h_j$ more_general_than_or_equal_to $h_k$(记作$h_j \ge_g h_k$), 当且仅当：
$$(\forall x \in X) [(h_k(x) = 1) \rightarrow (h_j(x) = 1)]$$

定义：一个假设$h$与训练集合$D$一致，当且仅当$D$中的每一个样例$(x, c(x))$,都有$h(x) = c(x)$
$$Consistent(h, D) \equiv (\forall \in D) [h(x) = c(x)]$$

定义：变型空间，$VS_{H,D}$
$$ VS_{H,D} = { h \in H | Consistent(h, D)} $$

变型空间，可使用一般边界(general boundary)和特殊边界(specific boundary)进行界定。

归纳偏置

无偏的学习器： 包含所有可能目标概念的假设空间，它是实例空间$X$的幂集($X$的所有子集的集合)

无偏学习器的无用性：对于任意未见的实例，它无法给出明确判断，即没有泛化能力。对于任一未见的实例$x$, 它总能刚好被变型空间中的一半假设判定为正例，一半假设判定为负例，无法通过投票决定结果。

归纳偏置：学习器如果不对目标概念做预先的假定，它从根本上无法对未见的实例进行分类，这些预先假定的前提集合称之为归纳偏置。

  • 归纳系统添加 归纳偏置之后，等价于某种形式的演绎系统
  • 不同学习器有不同的归纳偏置，通过比较归纳偏置，来比较不同学习器的泛化策略；有偏性越强的学习器，归纳能力越强，可以分类更多的未见实例。（可以分类，但不代表分类正确）

阿桑奇自传

《阿桑奇自传：不能不说的秘密》

@(读书笔记)[阿桑奇, 维基解密, 传记]

惊鸿一瞥

  • 保守势力会为我们戴上各种帽子，让媒体发动攻击，祭出“国家利益”、“爱国主义”的大旗，指控我们卖国叛变，但我们一直都深知，世界要比他们想象中的现代得多。开罗在等待着，突尼斯在等待着，我们都在等待着技术进步为全人类带来更大自由的那一天。未来世界中，力量不是来自于枪杆子，而是来自于交流通讯，人不会再通过得到一小群统治精英的认可来认识自己，而是在融入社交网络、充分挖掘巨大政治潜能的过程中认识自己。
    我以前一直相信，现在也深信不疑，“暴虐势力的主要力量源泉就是它能够秘密地行使权力。”

  • 维基解密的创立理念，就是源的存在应是可以无限否认的。我期待有一天，即使权贵势力威胁要惩罚所有张嘴说话的人，世人还是能够运用该技术大声说出来。

  • 我们对知识有一种责任，我们对信息有一种义务。正因为如此我们才对图书馆钟情有加。在数字时代，我们也许同样会明白，我们最终要承担责任，利用科技反抗那些阻止知识或信息公开传播的人。我们不能全靠报纸，因为事实一次又一次证明，报纸既是审查者，又有倾向性。我们不能信任广播，因为大多数情况下，广告的价值比新闻的价值对他们更有吸引力。因此，计算机时代的出版本质在于在系统允许下完成任务，同时抵制根深蒂固、自我保护的旧有出版惯例。

  • 美国总统西奥多·罗斯福就良政话题有一段箴言。他写道：“表层政府的后面有一个隐形政府，不忠于人民，也不为人民负责。摧毁隐形政府，斩断腐朽商界和腐朽政界的邪恶纽带，是政治家的首要任务。”

  • 当受压迫的媒体机构迫切关注、但自己又不能报导的某个重大事件被证实，被首先披露在国际媒体上时，这些受压迫的媒体就会得到解放。其它渠道都封死后，维基解密就是最后的披露者，同时也是一个外力不可阻挡的平台。

  • …需要找到一个开放性（而不是保密）的避风港才行。这样的一个地方也应该是新闻的避风港，由法律规定保护消息来源。新闻自由将是法律的基本精神，网络自由将是现代精神的一部分，免予起诉将是一项标准。我对这个避风港的展望渐渐成形：一个政治独立的地区，一个人们可以不再逃亡的地方，一个将举报人视为英雄而不是公敌的地方，一个法律咨询随时免费提供、全民上网的地方。(阿桑奇觉得冰岛可能合适。)

  • …(冰岛RUV电视台最后一刻接到法律禁制令，不能播放银行业丑闻…于是播放员：)他说，我们现在无法为您献上报导，但有一个组织可以。随后节目就在屏幕上大大打出了维基解密的标识，整整放了几分钟，填补原先留给被禁报导的时间。

三言两语

  • “一砖头扔下去”，估计砸到的都是像我这样不明白国家、政府、政党的意义和关系的人；不明白自由是什么。我们也不被鼓励讨论这些。我们整天讨论的是房子，也甘愿被房子束缚，也被那些有房子的人指责“整天就只会讨论房子”。我们不明白自己拥有什么，将拥有什么，不知道社会契约论、政府论；对世界历史一无所知。在时代的大潮中，我们卑微地蜷缩成了草履虫。

    万物皆比特。信息的大潮温柔又狂暴无匹，凡是阻截它的，都被它冲刷干净。

穷查理宝典

《穷查理宝典》

早先囫囵吞枣读完了一遍《穷查理宝典》，略作笔记，后续细读。读这些书倒不是想挖掘一两条投资宝典（也没钱投资），而是一种纯粹的好奇。下面记录了该书反复提到的一些知识。

心理学

查理非常看重心理学，看重人的心理和行为对于商业的影响。查理不喜欢宏观的泛泛而谈，更倾向细致入围观经济学。

  • 批评传统的心理学，过于呆板，脱离实际。比如“嫉妒”这一最常见的心理学活动都没有纳入教材
  • 收集和整理了20多个心理错误倾向，包括
    • 被剥夺超级反应倾向：即将到嘴的肉飞了
    • 奖励和惩罚超级反应倾向：有效的激励，避免被钻漏洞
    • 避免疑虑倾向：尽快做出决定来消除疑虑的倾向，容易武断行事。
    • 避免不一致倾向：大脑的快速反应和守旧。
    • 康德式公平倾向：对公平的期待。
    • 艳羡和妒忌倾向：人之常情，合理控制，驱动进步。
    • 受简单联想影响倾向：如价格和商品质量。品牌效应等。
    • 简单的、避免痛苦的心理否认：自我保护，但可能导致堕落。
    • 自视过高倾向：高估自己的能力和价值，无来由的固执，对团队危害极大。
    • 社会认同倾向：从众和趋同心理，当人困惑和压力重重时最容易发生。
    • 对比错误反应倾向：购物时虚报高价，抬高心理预期再缓慢砍价；租房时先给看很差的房子，再看普通房子。
    • LOLLAPPALOOZA效应：多种因素共同作用产生的更激进的效应。

跨学科

查理认为世界是复杂的，需要学习多学科（在他眼里，如有可能，应该是所有学科）的基本思想和思维模型，比如数学、物理学、心理学等。综合的看待问题，从而避免“铁锤人问题”：在手握铁锤的人眼中，所见的都是钉子。

  • 掌握多学科的主要模型。不需要成为每一个领域的专家，但了解每一个模型的出处和核心思想，构建多元思维模型。
  • 保持自主和常识。在多元思维下，保证自己的理性判断。
  • 终生学习。保持谦卑，终生学习，不断增加自己的模型清单。

习惯

  • 专注。专注做好一件事，不要反复无常。一次性把事情做对。
  • 集中投资。不要反复进行交易，徒然损失交易费；花绝大部分精力在评估企业资产的优劣上，寻找被低估的或者有极大发展潜力的股票。
  • 列出自己的投资原则检查清单。做重大决策时照此检查。
  • “祖母的规矩”：你们必须先吃完胡萝卜，然后才能吃甜点

提及的著名人物

  • 西塞罗：罗马的元老院领袖，诗人、哲学家。公共服务是罗马公民的最高义务。
  • 杰克·韦尔奇：通用电气CEO，《赢》
  • 本杰明·格拉汉姆：《证券分析》，《聪明的投资者》
  • 山姆·沃尔顿：沃尔玛CEO
  • 沃伦·巴菲特：伯克希尔·哈撒韦CEO
  • 亚当·斯密：《国富论》
  • 史蒂芬·平克：《语言本能》
  • 斯坦利·米尔格拉姆：道德信念与权威实验
  • 菲利普·费舍尔：《怎样选择成长股》
  • 罗杰·费舍尔：《谈判力》

当我们谈论Ax时在谈论什么

一、从解方程的角度

我们经常需要求解$Ax=b$,(这里$A是m*n维矩阵，x是n维列向量$)。我们可以对$A$（或者增广矩阵$[A, b]$）进行行变换得到阶梯型（归一化阶梯型），其中最关键的概念是pivot column, pivot position, free variable

  1. 当b = 0时，$A$满足什么条件， 解空间是非平凡的？

    • 当$A$的free variable数量不为0，即$A$的pivot column数量少于n； free variable的数量是解空间的维度。

  2. 当$b \ne 0$ 时， $A$满足什么条件，存在解？解是否唯一？对任意的b,是否都存在解？

    • 解的存在性：当且仅当增广矩阵的阶梯型，最后的非零行不是[0, b*]的形式。
    • 解的唯一性：在存在的前提下，若pivot的数量正好是n，那么就有唯一解;否则解不唯一
    • 若要对任意的b都存在解：$A$的每一行都存在一个pivot; 另一方面，说明$Ax$能够张成空间$R^m$, $rank(A) = m$。
  3. 对于非齐次方程，可以表达成特解与齐次解的和。
  4. 利用LU分解，方便解方程 $$Ax = b \Rightarrow Lz = b, Ux = z$$

二、从线性映射的角度

  1. $Ax$ 表示从$R^n \rightarrow R^m$的线性映射 $T(x): x \rightarrow Ax$, 其中 $A = [T(e_1), T(e_2)… T(e_n)]$
  2. A的分类
    • 满射(mapping onto): 对于$\forall b \in R^m, Ax = b$都有解，即$col A = R^m$；也即$rank A = m(m <= n)$，每一行都有一个pivot position.
    • 单射(one to one): $x \ne y \Rightarrow Ax \ne Ay $，等价于$Ax = 0$只有平凡解，即没有free variable, $rank A = n( n <= m)$, 每一列都是Pivot column.
      从基的角度来看，$ A$的各列是$col A$的一组基, 而不同的坐标$x$，自然对应不同的值。
    • 双射(bijection)： A 既是单射又是满射，则A一定是方阵，且$rank(A) = n$。
  3. 若$T: V - >W$ , 对应的变换矩阵$A$是双射，那么，$V和W同构$，从线性代数的角度来看，二者没有区别。另，若选定一组基，任何向量都可以映射（双射）到$R^n$， 因此n维向量空间都是同构的，我们可以用$R^n$来研究任何向量空间。$\forall b, Ax = b$存在唯一的解。
  4. 当$A$是方阵时，下面的结论是等价的：
    • $A$ 可逆
    • $detA \ne 0 $
    • $rank (A) = n, Null (A) = 0$
    • $\forall b, Ax = b $有唯一的解, 即 $col A = R^n$
    • $A$恰好有n个pivot column.
    • $A$ 是双射
    • $A$ 可以写成一系列初等矩阵的乘积，也就是行等价于单位矩阵E。

三、从最小二乘角度

  1. $Ax$不是满射, ($\exists b, Ax = b 无解$)， 相应的问题转换成最小二乘问题。
  2. 最小二乘问题：即 $argmin_{x}||b-Ax||$
    • 从线性空间角度来看，等价于求$b$在空间$Ax$的投影$b_0$, 最小取值为 $||b-b_0||$时的$x$
    • 从统计的角度来看，若噪音满足高斯分布，那么最小二乘问题等价于极大似然。
    • 当A为单射(rank A = n, 即A的各列线性无关)，最小二乘问题有唯一解，且x为$(A^TA)^{-1}A^Tb$ $(normal \hspace{0.2cm} equation)$
    • 当A不是单射时，normal equation中的$(A^TA)^{-1}$不存在（因为$rank A^TA = rank A < n$, 不可逆）； A对噪声变得敏感，噪声可能导致数值解法产生的结果相差较大；用机器学习的术语，variance过大。从另一个角度来讲，$L_2$正则化等价于$(A^TA + \lambda I)^{-1}A^Tb$

四、从线性动力系统角度：$x_{k+1} = Ax_k$

  1. 对角化

    • $A$对角化的充要条件：存在n个线性无关的特征向量构成P的列，则 $A = PDP^{-1}$，其中D为对角矩阵，元素为A的特征值。
    • 对于$T: x \rightarrow Ax$, 若以P的列向量为基，那么在T在P下的变换矩阵为D。即：$$ T_p: x_p \rightarrow Dx_p $$，线性变换将更直观方便，尤其涉及迭代时,可惜，并非所有矩阵A都可以对角化
    • 若A是实对称矩阵，那么A可以对角化，且P是正交矩阵；A的所有特征值都是实数。
    • 对于所有矩阵A（并不限定是方阵）都存在SVD分解；
      $$ A = U \Sigma V $$
      其中 $U，V$都是正交矩阵，$\Sigma$是对角矩阵，对角线的值为奇异值（$A^TA$的特征值的平方根），通过截取部分绝对值较大的奇异值，以较小的能量损失为代价，减小矩阵规模。

  2. 线性动力系统

    • 典型例子是马尔科夫链
    • 基于特征空间的分解：若A有n个线性无关的特征向量构成基$[v_1, v_2…v_n]$,相应的特征值$[\lambda_1, \lambda_2…\lambda_n]$(假设绝对值降序排列), 将初始值写作:$$ x_0 = c_1v_1 + c_2v_2 + … + c_nv_n $$
      那么 $$ x_k = A^kx_0 = c_1\lambda_1^kv_1 + c_2\lambda_2^kv_2 + … + c_n\lambda_1^kv_n $$
      $x_k$收敛，当前仅当 $|\lambda_i| \le 1$;
    • 马尔科夫链的状态矩阵A一定有特征值1，若A是regular的，那么 $x_k$收敛到稳态
    • 事实上，这里的理论非常丰富有趣，一个有趣的例子是一阶线性微分方程组；之前学习通信理论，比如求解电路状态时的种种不适，在这里得到解答。参看Linear Algebra and Its Application, chapter 5.7
