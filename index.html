当我们谈论Ax时在谈论什么

一、从解方程的角度

我们经常需要求解$Ax=b$,(这里$A是m*n维矩阵，x是n维列向量$)。我们可以对$A$（或者增广矩阵$[A, b]$）进行行变换得到阶梯型（归一化阶梯型），其中最关键的概念是pivot column, pivot position, free variable

  1. 当b = 0时，$A$满足什么条件， 解空间是非平凡的？

    • 当$A$的free variable数量不为0，即$A$的pivot column数量少于n； free variable的数量是解空间的维度。

  2. 当$b \ne 0$ 时， $A$满足什么条件，存在解？解是否唯一？对任意的b,是否都存在解？

    • 解的存在性：当且仅当增广矩阵的阶梯型，最后的非零行不是[0, b*]的形式。
    • 解的唯一性：在存在的前提下，若pivot的数量正好是n，那么就有唯一解;否则解不唯一
    • 若要对任意的b都存在解：$A$的每一行都存在一个pivot; 另一方面，说明$Ax$能够张成空间$R^m$, $rank(A) = m$。
  3. 对于非齐次方程，可以表达成特解与齐次解的和。
  4. 利用LU分解，方便解方程 $$Ax = b \Rightarrow Lz = b, Ux = z$$

二、从线性映射的角度

  1. $Ax$ 表示从$R^n \rightarrow R^m$的线性映射 $T(x): x \rightarrow Ax$, 其中 $A = [T(e_1), T(e_2)… T(e_n)]$
  2. A的分类
    • 满射(mapping onto): 对于$\forall b \in R^m, Ax = b$都有解，即$col A = R^m$；也即$rank A = m(m <= n)$，每一行都有一个pivot position.
    • 单射(one to one): $x \ne y \Rightarrow Ax \ne Ay $，等价于$Ax = 0$只有平凡解，即没有free variable, $rank A = n( n <= m)$, 每一列都是Pivot column.
      从基的角度来看，$ A$的各列是$col A$的一组基, 而不同的坐标$x$，自然对应不同的值。
    • 双射(bijection)： A 既是单射又是满射，则A一定是方阵，且$rank(A) = n$。
  3. 若$T: V - >W$ , 对应的变换矩阵$A$是双射，那么，$V和W同构$，从线性代数的角度来看，二者没有区别。另，若选定一组基，任何向量都可以映射（双射）到$R^n$， 因此n维向量空间都是同构的，我们可以用$R^n$来研究任何向量空间。$\forall b, Ax = b$存在唯一的解。
  4. 当$A$是方阵时，下面的结论是等价的：
    • $A$ 可逆
    • $detA \ne 0 $
    • $rank (A) = n, Null (A) = 0$
    • $\forall b, Ax = b $有唯一的解, 即 $col A = R^n$
    • $A$恰好有n个pivot column.
    • $A$ 是双射
    • $A$ 可以写成一系列初等矩阵的乘积，也就是行等价于单位矩阵E。

三、从最小二乘角度

  1. $Ax$不是满射, ($\exists b, Ax = b 无解$)， 相应的问题转换成最小二乘问题。
  2. 最小二乘问题：即 $argmin_{x}||b-Ax||$
    • 从线性空间角度来看，等价于求$b$在空间$Ax$的投影$b_0$, 最小取值为 $||b-b_0||$时的$x$
    • 从统计的角度来看，若噪音满足高斯分布，那么最小二乘问题等价于极大似然。
    • 当A为单射(rank A = n, 即A的各列线性无关)，最小二乘问题有唯一解，且x为$(A^TA)^{-1}A^Tb$ $(normal \hspace{0.2cm} equation)$
    • 当A不是单射时，normal equation中的$(A^TA)^{-1}$不存在（因为$rank A^TA = rank A < n$, 不可逆）； A对噪声变得敏感，噪声可能导致数值解法产生的结果相差较大；用机器学习的术语，variance过大。从另一个角度来讲，$L_2$正则化等价于$(A^TA + \lambda I)^{-1}A^Tb$

四、从线性动力系统角度：$x_{k+1} = Ax_k$

  1. 对角化

    • $A$对角化的充要条件：存在n个线性无关的特征向量构成P的列，则 $A = PDP^{-1}$，其中D为对角矩阵，元素为A的特征值。
    • 对于$T: x \rightarrow Ax$, 若以P的列向量为基，那么在T在P下的变换矩阵为D。即：$$ T_p: x_p \rightarrow Dx_p $$，线性变换将更直观方便，尤其涉及迭代时,可惜，并非所有矩阵A都可以对角化
    • 若A是实对称矩阵，那么A可以对角化，且P是正交矩阵；A的所有特征值都是实数。
    • 对于所有矩阵A（并不限定是方阵）都存在SVD分解；
      $$ A = U \Sigma V $$
      其中 $U，V$都是正交矩阵，$\Sigma$是对角矩阵，对角线的值为奇异值（$A^TA$的特征值的平方根），通过截取部分绝对值较大的奇异值，以较小的能量损失为代价，减小矩阵规模。

  2. 线性动力系统

    • 典型例子是马尔科夫链
    • 基于特征空间的分解：若A有n个线性无关的特征向量构成基$[v_1, v_2…v_n]$,相应的特征值$[\lambda_1, \lambda_2…\lambda_n]$(假设绝对值降序排列), 将初始值写作:$$ x_0 = c_1v_1 + c_2v_2 + … + c_nv_n $$
      那么 $$ x_k = A^kx_0 = c_1\lambda_1^kv_1 + c_2\lambda_2^kv_2 + … + c_n\lambda_1^kv_n $$
      $x_k$收敛，当前仅当 $|\lambda_i| \le 1$;
    • 马尔科夫链的状态矩阵A一定有特征值1，若A是regular的，那么 $x_k$收敛到稳态
    • 事实上，这里的理论非常丰富有趣，一个有趣的例子是一阶线性微分方程组；之前学习通信理论，比如求解电路状态时的种种不适，在这里得到解答。参看Linear Algebra and Its Application, chapter 5.7

hexo 博客搭建指南

Install hexo

若npm很慢，请将npm的源换成taobao的。
如果出现问题，可以参见这篇文章：hexo 安装与配置

Initial hexo folder

进入一个空文件夹，执行

Run server

默认打开本地的4000端口，可以用于调试。可以马上看到hexo的hello world页面

More info: Server

Create a new post

新建文章到 source/_posts/xxx.md, 通过相应md编辑器编辑即可。

More info: Writing

Link to github

  • 打开_config.yml, 在最后deploy 字段编写如下：

  • 该github的仓库名，必须命令为 your_github_name.github.io

  • 保证 npm install hexo-deployer-git –save 执行成功；

Generate static files

生成静态文件到public目录，在发布之前必须先进行这个操作

More info: Generating

Deploy to remote sites

将文章部署到远程仓库，比如github。 在_config.yml中“deploy”进行配置。

More info: Deployment

Custom domain

  • 购买域名
  • 进入相应工程，创建名为CNAME的文件，文件内容为域名
  • 工程右上角有个”Settings”, 进入之后,在“custom domain”处填写域名
  • 在域名商那里设置A record, 或者CNAME record。

Support math

  • 执行操作: npm install hexo-math –save

  • 在_config.yml中添加 plugins: hexo-math

    行内公式： $e^{i\pi} + 1 = 0$
    行间公式： $$A = U \Sigma V$$

Tips

  • hexo的命令一般可以用首字母缩写代替，比如 hexo server = hexo s
    hexo deploy –generate = hexo d -g
  • 新建文章，并发布的操作一般是：
    • hexo n “your_article_title”
    • hexo clean
    • hexo d -g
  • 在本地调试，不需要进行generate 和deploy.
